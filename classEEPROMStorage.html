Detailed Description

template<typename T>
class EEPROMStorage< T >

Provides direct access to an EEPROM variable.

This class provides direct access to an EEPROM variable wrapped as type T. Accessing the value of the variable is done directly from the EEPROM variable. Subsequently, writing the variable value is done directly to the EEPROM memory.

Template Parameters
TThe type of the variable stored.
